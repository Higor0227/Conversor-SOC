## Este programa automatiza a criação de documentos de cotação de insumos, gerando os seguintes arquivos:

### IN.pdf: Documento principal contendo a IN (Instrução Normativa) com as cotações em anexo.
### sga.txt: Arquivo de texto com as informações formatadas para inserção no sistema SGA.
### data.txt: Um arquivo de backup com todos os dados inseridos, permitindo a importação para ajustes futuros no programa.

## Como Usar
Siga estes 5 passos para gerar seus documentos de cotação:

1. Dados Principais: Preencha as informações do insumo, item e unidade na seção superior do programa.
2. Dados dos Fornecedores: Insira os dados de cada fornecedor nos campos correspondentes.
3. Anexar Cotações: Arraste e solte os arquivos de cotação (PDFs, imagens, etc.) na área indicada ou clique em "Procurar..." para selecioná-los manualmente.
4. Diretório de Saída: Especifique a pasta onde os documentos finais serão salvos.
5. Gerar Arquivos: Clique em "CRIAR DOCUMENTO FINAL" para iniciar o processo de automação.

## Dicas
1. Limpar Cache: Utilize o botão "Limpar Cache" para remover arquivos temporários que possam ter sido deixados para trás caso o programa seja fechado inesperadamente. Isso garante o bom funcionamento da aplicação.
2. Importar Dados: O programa exporta um arquivo chamado data.txt. Para agilizar o preenchimento, você pode usar o botão "Importar" e selecionar um arquivo data.txt de uma cotação anterior para carregar todos os dados automaticamente.
